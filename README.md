# pycorrect

[![Build Status](https://travis-ci.org/jml/pycorrect.svg?branch=master)](https://travis-ci.org/jml/pycorrect)

Analyzes Python code for correctness.

## Contributing

Build the project:

```
stack build
```

Run the binary:

```
stack exec pycorrect
```

Run the tests:

```
stack test
```

Run the benchmarks:

```
stack bench
```

Generate documentation:

```
stack haddock
```
